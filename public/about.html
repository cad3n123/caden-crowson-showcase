About

Personal portfolio of Caden Crowson

Hi, I'm Caden Crowson. I'm a software developer who enjoys building small, interactive web experiences — simulations, generative tools, and the occasional oddball utility. This site collects the projects I've put out into the world so people can try them without installing anything.

What's on this site

The home page is styled as a retro television: each "channel" is a live preview of one of my projects, and each VHS tape on the rack swaps the channel. Clicking Launch Site takes you to the full version of whichever project is currently on screen.

How it's built

The showcase shell is a Vite + React app. Each project lives in its own repository and is deployed independently to Vercel; the showcase domain uses subpath rewrites so they all live under one URL. The code is hand written — no template, no site builder.
